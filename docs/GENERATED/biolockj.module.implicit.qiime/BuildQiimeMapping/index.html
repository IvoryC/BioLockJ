BuildQiimeMapping#

Add to module run order:
#BioModule biolockj.module.implicit.qiime.BuildQiimeMapping

This page is a place holder.
This module does not have a properly generated user guide page because it does not implement the ApiModule interface.
There may be a manually created page elsewhere.

Adds modules#

pre-requisite modules
none found

Docker#

If running in docker, this module will run in a docker container from this image:

biolockjdevteam/qimme_classifier:v1.3.9

This can be modified using the following properties:
BuildQiimeMapping.imageOwner
BuildQiimeMapping.imageName
BuildQiimeMapping.imageTag
